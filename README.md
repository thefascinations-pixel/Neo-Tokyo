# Neo Tokyo District

A lightweight browser exploration sim prototype built with plain HTML, CSS, and Canvas.

## Run

Open `index.html` in a browser, or serve the folder with any static file server.

## Controls

- `WASD` or arrow keys: move
- click or tap the world: click-to-move pathing
- `Space`: focus the nearest landmark
- `R`: use a nearby roof access point
- `M`: start or mute the `Neon Rain Reverie.mp3` soundtrack
- mobile: left thumbstick plus `Focus`, `Roof`, and `Audio` buttons

## Prototype Scope

This build contains two connected Neo Tokyo blocks rendered in an isometric 3D style:

- neon skyline backdrop
- rainy streets, plazas, and boulevard reflections
- tile-based collision so visually open paths are walkable
- rooftop access points and short skybridges
- smoother camera easing and velocity-based movement
- click-to-move and mobile touch controls
- wandering NPC pedestrians
- landmark discoveries with ambient story text
- looping MP3 soundtrack: `Neon Rain Reverie.mp3`
